# 🍽️ Zomato Restaurant Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Tableau Dashboard** that analyzes Zomato restaurant data from multiple countries. The dashboard provides insights into restaurant distribution, customer ratings, pricing, online delivery availability, restaurant chains, and country-wise performance to help understand restaurant trends and customer preferences.

---

## 🎯 Objectives

- Analyze restaurant distribution across different countries.
- Compare average restaurant ratings by country.
- Study the average cost for two people across locations.
- Identify top restaurant chains.
- Analyze online delivery availability.
- Visualize restaurant cost distribution.
- Explore the relationship between restaurant cost and ratings.
- Identify local favorite restaurants.

---

## 📊 Dashboards Included

### 1. Summary Dashboard
Provides an overview of:
- Total Countries
- Total Restaurants
- Average Rating
- Overall Restaurant Statistics

### 2. Restaurant Dashboard
Includes visualizations for:
- Number of Restaurants by Country
- Average Cost by Country
- Average Rating by Country
- Cost Distribution
- Cost vs Rating (Trend Line)
- Top 20 Restaurant Chains
- Restaurants Supporting Online Delivery
- Local Favorite Restaurants

### 3. Customer Dashboard
Focuses on customer-related insights such as:
- Ratings
- Delivery Performance
- Restaurant Popularity
- Customer Preferences
